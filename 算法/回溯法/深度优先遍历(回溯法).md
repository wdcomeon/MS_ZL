# 括号生成
数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

 

示例 1：

输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]
示例 2：
输入：n = 1
输出：["()"]
```python3
class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        res=[]
        cur=""
        def dfs(cur,left,right):
            if not left and not right:
                res.append(cur)
                return 
            if right<left:
                return 
            if left>0:
                dfs(cur+"(",left-1,right)
            if right>0:
                dfs(cur + ")",left,right-1)
        dfs(cur,n,n)
        return res
            
```
